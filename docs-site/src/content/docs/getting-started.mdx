## Quick Start (Local)

### 1. Scaffold your project

```bash
npx bikinibottom init my-reef
cd my-reef
```

This creates `ORG.md` (your agent org definition) and `bikinibottom.config.json` (server config).

### 2. Start the server

```bash
npx bikinibottom start
```

Open [http://localhost:3333](http://localhost:3333) â€” your dashboard is live.

### 3. Discover your agents via A2A

<Tabs>
  <TabItem label="bash">
    ```bash
    curl http://localhost:3333/.well-known/agent.json
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    import httpx
    card = httpx.get("http://localhost:3333/.well-known/agent.json").json()
    print(f"Agent: {card['name']} â€” {len(card['skills'])} skills")
    ```
  </TabItem>
  <TabItem label="TypeScript">
    ```typescript
    const card = await fetch("http://localhost:3333/.well-known/agent.json").then(r => r.json());
    console.log(`Agent: ${card.name} â€” ${card.skills.length} skills`);
    ```
  </TabItem>
</Tabs>

**Response:**

```json
{
  "name": "My Reef",
  "url": "http://localhost:3333",
  "protocolVersion": "0.3",
  "capabilities": { "streaming": true },
  "skills": [{ "id": "task-delegation", "name": "Task Delegation" }]
}
```

### 4. Send a task

```bash
curl -X POST http://localhost:3333/a2a/message/send \
  -H 'Content-Type: application/json' \
  -d '{
    "message": {
      "role": "user",
      "parts": [{ "kind": "text", "text": "Build a REST API for user management" }]
    }
  }'
```

### 5. Use as MCP tool server

```bash
curl -X POST http://localhost:3333/mcp \
  -H 'Content-Type: application/json' \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/list","params":{}}'
```

## Configuration

`bikinibottom.config.json` controls your server:

```json
{
  "port": 3333,
  "orgFile": "ORG.md",
  "simulation": { "mode": "deterministic", "tickInterval": 3000 },
  "router": { "preferLocal": true, "providers": ["ollama", "groq"] },
  "protocols": { "a2a": true, "mcp": true }
}
```

---

## Troubleshooting

**Port already in use?** Change it in `bikinibottom.config.json`: `{ "port": 3334 }`

**Ollama not available?** BikiniBottom falls back to Groq or OpenRouter automatically:

```bash
export GROQ_API_KEY=your_key
export OPENROUTER_API_KEY=your_key
```
