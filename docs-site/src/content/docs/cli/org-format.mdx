## Sections

### Identity

The `## Identity` section sets org-level metadata:

```markdown
## Identity
- Name: My Organization
- Mission: What this org does
```

### Structure

The `## Structure` section contains agent definitions. Each agent is an H3 heading with `(Level N)`:

```markdown
### Agent Title (Level 7)
- Name: Display Name
- Domain: engineering
- Role: lead
```
